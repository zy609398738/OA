Class TableView

用于显示数据表格的JavaScript控件. 集成的分页控件, 可对表格中的数据集进行客户端分页.

Class Detail
TableView(id:)

Author: ideawu@163.com. 
### HTML代码:
<div id="my_div"></div>

### JavaScript代码:
var table = new TableView('my_div');
table.dataKey = 'id';
table.header = {
	'id' : 'Id',
	'name' : 'Name',
};

table.add({id:1, name:'Tom'});
table.render();
Parameters:
{String} id:
HTML节点的id, 控件将显示在该节点中.
Returns:
{TableView} : 返回分页控件实例.
Requires:
jQuery PagerView
